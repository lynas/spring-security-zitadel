# Add ID provider Microsoft Login to Zitadel

- Add a new app in Azure AD
- https://portal.azure.com/#view/Microsoft_AAD_RegisteredApps/ApplicationsListBlade
- Create a app and get the client id and client secret
- Steps to add Microsoft Login to Zitadel
- https://zitadel.com/docs/guides/integrate/identity-providers/azure-ad#create-a-new-azure-ad-provider
- Open Zitadel instance console in instance (user must be admin to see it)
- https://test-fojq5t.zitadel.cloud/ui/console/settings?id=idp
- Click on Microsoft and client id and client secret
- Activate it from the list by clicking the check mark
- Go to the setting in organization level and activate it there as well
